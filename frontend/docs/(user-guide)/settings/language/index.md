By default, the RAWeb web client uses each user's browser language preference to determine which language to display the interface in. This policy overrides that preference and forces all users to see the interface in a specific language.

## Changing the interface language

1. Open the RAWeb app and go to the **Settings** page.
2. Find the **Language** section.
3. Choose a different language from the dropdown menu.

<InfoBar>

This setting will be saved in your browser's local storage and will persist across sessions. If you clear your browser's local storage or use a different browser, you will need to set the language again.

</InfoBar>

<InfoBar severity="caution">

Your administrator may have configured a forced language policy that overrides this setting. If the language dropdown is disabled or missing, contact your administrator for assistance.

</InfoBar>

If you are interested in contributing translations for additional langauges, please review [Translating RAWeb](https://github.com/kimmknight/raweb/blob/master/TRANSLATING.md).
