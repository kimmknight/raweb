---
title: Icon backgrounds
---

When icon backgrounds are enabled, RAWeb adds a square background with padding behind each app and desktop icon. This can make icons easier to distinguish when the icon itself has a transparent background or blends into the card color.

Icon backgrounds are enabled by default and only affect the [card view](/docs/view-modes/). Icons in grid, tile, and list views are not affected and never show icon backgrounds.

## Enabling or disabling icon backgrounds

1. Open the **Settings** page.
2. Find the **Icon backgrounds** section.
3. Toggle **Enable icon backgrounds** on or off.

<InfoBar>

If your administrator has configured this setting as a policy, the toggle will be disabled and you will not be able to change it.

</InfoBar>
