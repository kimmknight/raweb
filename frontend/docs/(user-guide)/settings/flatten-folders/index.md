---
title: Flatten folders
---

Apps and desktops in RAWeb can be organized into virtual folders. When flatten folders is enabled, RAWeb ignores the folder hierarchy and displays all resources together in a single flat list instead. This is useful if you prefer not to navigate through folders to find a resource.

Flatten folders is disabled by default.

## Enabling or disabling flatten folders

1. Open the **Settings** page.
2. Find the **Flatten folders** section.
3. Toggle **Enable flat mode** on or off.

<InfoBar>

Flatten folders is also supported in [simple mode](/docs/simple-mode/).

</InfoBar>

<InfoBar>

If your administrator has configured this setting as a policy, the toggle will be disabled and you will not be able to change it.

</InfoBar>
